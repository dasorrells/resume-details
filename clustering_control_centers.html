Control Center Assignment Process

“Developed an algorithm to systematically assign control centers to a control group using k-means clustering, geographic proximity, and other center maturity classifications.”

1. Data Preparation & Feature Engineering

  • Data Extraction: Extracted data using SQL queries and consolidated it into pandas DataFrames.
  • Aggregation: Calculated metrics like net sales, guest counts, and YOY variances over multiple timeframes (3M, 6M, 12M).
  • Attribute Joining: Added features such as geolocation, finance opening dates, and maturity classifications.

2. Feature Scaling & Normalization

  • Heavy Features: Scaled numeric features (e.g., total sales, guest counts) using MinMaxScaler mapped to [0, 1], multiplied by 2.
  • YOY Variances: Normalized percentage variances with MinMaxScaler mapped to [−1, 1].

3. Clustering Process

  • Cluster Determination: Used the elbow method to identify optimal clusters within maturity classes.
  • Label Assignment: Grouped centers into clusters based on operational similarities.

4. Control Center Assignment

  • Geographic Filtering: Selected nearest centers within the same maturity class.
  • Cluster-Based Selection: Identified additional centers with matching cluster labels.
  • Exclusion Criteria: Removed centers with significant price changes or unique promotions.

Bottom Line

This solution streamlines performance benchmarking by leveraging advanced clustering techniques and ensuring robust comparisons across lifecycle stages, geographic proximity, and YOY trends.
