Overview

This end-to-end solution identifies operationally similar control centers for comparison against target centers. Suitability is determined through clustering based on key metrics and lifecycle stages.

Data Extraction and Preparation

Key steps include:

  • Extracting data using SQL queries and consolidating into a pandas DataFrame.
  • Aggregating metrics such as net sales and guest counts over 3M, 6M, and 12M time windows.
  • Calculating year-over-year (YOY) percentage variances.
  • Joining additional attributes (e.g., geolocation, maturity classification).

Feature Scaling and Normalization

A dual-scaling approach is applied:

  • Heavy features: MinMaxScaler [0, 1], scaled further by 2.
  • YOY variance features: MinMaxScaler [âˆ’1, 1].

Clustering Process

The elbow method determines the optimal number of clusters. Centers are grouped into:

  • New
  • Ramping
  • Mature

Centers within the same cluster and maturity class are assigned labels for comparison.

Control Center Assignment

  1. Identify geographically nearest centers within the same maturity class.
  2. Select additional centers from the same cluster group.
  3. Exclude centers based on criteria (e.g., recent price changes).

Summary and Impact

This solution streamlines performance benchmarking, ensuring accurate comparisons using advanced clustering techniques.
