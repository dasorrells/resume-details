Marketing Attribution Modeling Framework

“Engineered a multi-model marketing attribution framework using LSTM and XGBoost to forecast guest acquisition and quantify channel impact, with SHAP-based explainability and temporal feature engineering.”

1. Goal

To build a replicable modeling pipeline that connects multi-channel media spend to downstream acquisition, leveraging temporal modeling and interpretability techniques to inform future marketing strategy.

2. Architecture & Inputs

  • Aggregated 12+ marketing channels into a daily panel structure
  • Engineered lagged features (e.g., trailing 3-day spend), 7-day trends, day-of-week one-hot encodings, and holiday indicators
  • Used guest count (USER_ID) as the primary forecasting target

3. Forecasting Models

  • LSTM Neural Network: Multi-step time series forecasting with 2-day input and 2-day horizon, trained in TensorFlow with dropout and early stopping
  • XGBoost Regression: Used for benchmarking and interpretability; evaluated via MAE, RMSE, R², and MAPE across train/test sets

4. Model Explainability

SHAP (SHapley Additive exPlanations) was applied to the trained XGBoost model to extract global and local feature importance:

  • Global: SHAP summary plots visualized which features consistently influenced guest acquisition
  • Local: Force plots helped diagnose how specific campaigns or conditions influenced single-day performance
  • Naive 95% confidence intervals were generated to contextualize forecast reliability

5. Strategic Use Cases

The framework is reusable for regional or campaign-level segmentation, with tunable lookback/horizon windows. It supports proactive media mix optimization and hypothesis testing for high-performing acquisition channels.

Bottom Line

This hybrid attribution framework provides both predictive accuracy and interpretability. It bridges marketing execution with analytics by quantifying impact, improving budget allocation, and surfacing key drivers behind acquisition performance.
