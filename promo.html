Promotional Analytics Measurement Framework

“Designed a repeatable measurement framework to evaluate promotional service offers, blending SQL and Python for holistic performance tracking across adoption, retention, and financial impact.”

1. Objective

Built a modular analytics solution to evaluate promotional campaign effectiveness—measuring incremental usage, post-offer retention, and revenue impact while distinguishing between new and returning guests.

2. Guest Cohort Classification

  • Defined qualifying users based on first-time redemption per service category
  • Segmented guest types into Net New, Qualified Re-activations, and Too Early to Qualify
  • Enabled filtering by service history lookback and dynamic qualification logic

3. Event Mapping and Conversion Tracking

The framework captures all monetizable interactions across a fixed follow-up window (e.g., 90 days), using flexible logic to include:

  • Service sales and discounted usage
  • Package redemptions and purchases
  • Cross-service guest movement post-offer

4. SQL & Python Synergy

SQL was used to define event logic, baseline classifications, and create layered CTEs. Python analytics followed to:

  • Summarize performance by guest cohort
  • Visualize adoption funnels and behavioral lift
  • Incorporate discount offsets and post-period cumulative value

5. Use Cases and Reusability

  • Framework supports iterative promotional A/B testing and ROI measurement
  • Easily adapted to new offers, body part targets, or test regions
  • Used to inform future promotional guardrails, targeting strategies, and offer design

Bottom Line

This framework provides a best-in-class approach to promotional analytics by integrating data engineering, SQL modeling, and Python-based behavioral analysis—empowering scalable measurement and strategic decision-making across a variety of offer types.
